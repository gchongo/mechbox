### 1.2 核心价值

**解决痛点：**
1. 手工计算尺寸链易出错，公式难记（T = L1 + L2 + L3...）
2. 不理解概率统计原理，无法进行 RSS 分析（T = √(ΣT²)）
3. 缺少可视化工具，难以直观理解尺寸链（箭头 + 标注）
4. 西格玛水平计算复杂，需要专业软件（Cpk = min(上下限)/3σ）

**核心价值：**
1. **可视化计算**：矢量图 + 公式 + 表格三合一展示
2. **步骤引导**：5 步法（选择类型→封闭环→组成环→方法→结果）
3. **概率统计**：完整统计模块（分布/转换/RSS/西格玛/Monte Carlo）
4. **防错机制**：自动判断增环/减环，实时验证输入
5. **教育工具**：教程 + 案例 + 练习题库

**目标效果：**
- 机械设计师：10 分钟完成一次装配间隙分析
- 质量工程师：批量验证 50 个零件公差合格性
- 学生：理解尺寸链原理，通过课程考试

```javascript
  normal: { k: 6.0, cv: 1.0, coverage: 0.9973, name: '正态分布' },

  triangular: { k: 4.24, cv: 0.707, coverage: 0.95, name: '三角分布' },
  skewed: { k: 5.0, cv: 0.8, coverage: 0.98, name: '偏态分布' }
};

// 2. 公差→标准差转换
function toleranceToSigma(tolerance, distribution = 'normal') {
  return tolerance / DISTRIBUTIONS[distribution].k;
}

// 3. 标准差→公差转换
function sigmaToTolerance(sigma, distribution = 'normal') {
  return sigma * DISTRIBUTIONS[distribution].k;
}

// 4. 极值法（最坏情况）
function worstCaseMethod(rings) {
  return rings.reduce((sum, ring) => sum + ring.tolerance, 0);
}

// 5. RSS 法
function rssMethod(rings) {
  const sumSquares = rings.reduce((sum, ring) => sum + ring.tolerance ** 2, 0);
  return Math.sqrt(sumSquares);
}

// 6. 西格玛水平
function calculateSigmaLevel(tolerance, sigma) {
  return tolerance / (6 * sigma);
}

// 7. Cpk
function calculateCpk(upperSpec, lowerSpec, mean, sigma) {
  const cpu = (upperSpec - mean) / (3 * sigma);
  const cpl = (mean - lowerSpec) / (3 * sigma);
  return Math.min(cpu, cpl);
}

// 8. 合格率
function calculatePassRate(sigmaLevel) {
  return 2 * cdfNormal(sigmaLevel) - 1;
}

// 9. DPPM
function calculateDPPM(passRate) {
  return (1 - passRate) * 1000000;
}
```

```nginx
server {
    server_name mechbox.cax.do;
    location / {
        root /var/www/mechbox;
        try_files $uri $uri/ /index.html;
    }
    ssl_certificate /etc/letsencrypt/live/mechbox.cax.do/fullam;
```

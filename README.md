# MechBox 机械工具箱

面向机械设计、质量与工艺人员的 **Web 工程计算平台**，以尺寸链与公差叠加分析为核心，并集成概率统计、蒙特卡洛模拟及常用机械强度校核工具。

> **使用说明**：本工具输出基于简化工程模型，仅供学习、估算与辅助分析，不能替代完整标准复核或正式设计签核。涉及安全关键结构时，须由持证工程师按企业规范确认。

## 主要功能

### 尺寸链与公差

- **5 步引导式尺寸链编辑器**：选择分析类型 → 定义封闭环 → 添加组成环 → 选择叠加方法 → 查看结果
- **多种叠加方法**：极值法（Worst Case）、RSS、修正 RSS、6σ RSS
- **增/减环自动判断**，支持 mm / inch 单位切换
- **GD&T 公差栈**、**公差分配**、**批量尺寸链分析**
- **Monte Carlo 模拟**与编辑器数据联动
- 矢量示意图、公式推导、方法对比表、西格玛 / Cpk 摘要

### 概率统计

- 正态、均匀、三角、偏态等分布参数与转换
- 公差 ↔ 标准差、RSS 叠加、合格率与西格玛水平
- 假设检验、控制图等扩展统计工具

### 机械计算工具

涵盖但不限于：

| 类别 | 工具 |
|------|------|
| 连接与紧固 | 螺栓预紧力、螺栓组、螺纹、键连接、焊缝 |
| 传动与轴系 | 齿轮、轴承、轴（扭转/组合）、离合器、皮带 |
| 结构与强度 | 梁挠度、弹簧、疲劳、钣金、结构件 |
| 配合与热 | ISO 286 配合、过盈配合、热膨胀 |
| 液压与其它 | 液压/气缸、O 形圈、材料选型、制造工艺 |

各工具支持多计算模式（简化 / 完整 / 专业），结果可保存至本地历史并导出报告。

### 学习与参考

- 案例库、教程、练习题库、公式手册、术语词典、FAQ
- 工具地图与上下文帮助

### 数据与导出

- 分析结果保存至浏览器 **localStorage**（无需后端数据库即可使用核心功能）
- 支持 **PDF / Excel / PNG** 导出及历史记录合并报告
- 收藏、备份与恢复

## 技术栈

- **前端**：Vue 3、Vite、Vue Router
- **UI**：Element Plus、Tailwind CSS
- **图表与公式**：Plotly.js、KaTeX、math.js
- **导出**：jsPDF、html2canvas、SheetJS (xlsx)
- **测试**：Vitest

## 本地开发

```bash
npm install
npm run dev
```

浏览器访问开发服务器地址（默认由 Vite 输出，一般为 `http://localhost:5173`）。

## 构建与预览

```bash
npm run build
npm run preview
```

构建产物位于 `dist/`，可部署至任意静态 Web 服务器。

## 测试

```bash
npm test
```

## 项目结构

```
src/
├── assets/          # 静态资源与样式
├── components/      # 可复用组件（布局、计算页、图表等）
├── composables/     # 组合式 API
├── constants/       # 常量、案例、模板配置
├── i18n/            # 中英文文案
├── router/          # 路由
├── utils/           # 计算引擎、导出、存储
└── views/           # 页面视图
tests/               # 单元测试
```
